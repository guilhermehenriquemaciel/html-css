Lista de Tags HTML

Em HTML5 podemos inserir os elementos da lista a seguir. passe sobre os termos para descobrir qual a tag deverá utilizar em cada clase .

    • Cabeçalho <h1> - <h6>

    • Parágrafo <p>

    • Link <a>

    • Imagem <img>

    • Lista não ordenada <ul>

    • Lista ordenada <ol>

    • Item de lista <li>

    • Divisão <div>

    • Span <span>
